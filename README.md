# 🤖 Nova Chatbot

## 📌 Project Overview
Nova is a Rule-Based AI Chatbot built using Python.  
It simulates a conversational assistant with memory, intent detection, and personalized responses.

This project was developed as part of an AI internship to understand the basics of conversational AI systems.

---

## ✨ Features
- Greeting detection (Hello, Hi, Hey)
- Remembers user's name 🧠
- Personalized responses
- Help system with clear instructions
- Intent-based response system
- Friendly chatbot personality 🤖

---

## 🛠️ Technologies Used
- Python 3
- No external libraries required
